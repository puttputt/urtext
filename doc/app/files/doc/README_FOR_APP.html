URTextbook

How to install: First you need to install Ruby and Ruby Gems. Ruby and ruby gems installers/sources can be found rubyonrails.org/download Once RubyGems is installed run: sudo gem install -v=3.0.3 rails —include-dependencies

In the application directory run the command: bundle install This will install gems bundled with the application. If the bundle gem is not installed, you may need to run sudo gem install bundle.

You may run the application using WEBRICK webserver with: rails s
